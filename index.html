KanbanApp gratuito, privado e pronto para uso corporativo

O KanbanApp é 100% gratuito e foi pensado para equipes e empresas que precisam de um quadro Kanban simples, rápido e sem dependência de serviços externos.

O app não armazena dados pessoais dos usuários. Você pode usar online agora pelo botão Acessar app ou baixar o arquivo HTML para uso offline pelo botão Download.

Para usar offline, basta baixar o kanbanapp.html, abrir o arquivo no navegador e começar a organizar seu board. O conteúdo do board fica no cache/localStorage do navegador, então mantenha sempre um backup em JSON salvo na sua máquina.

Aviso: o KanbanApp não se responsabiliza por perda de dados causada por limpeza de cache do navegador. Para reduzir riscos, utilize regularmente a exportação em JSON e mantenha cópias de backup no seu computador.
